The Operating System

MintKit

MintKit is a custom Linux OS built for the PocketMint hardware on Raspberry Pi Zero 2 W. Open source. Built with Python + pygame. Boots straight to the launcher. Ships with autostart, udev button mapping, brightness and volume controls.

# Quick start on PC
$ pip install pygame
$ python3 mintos.py
 
# Build a hardware image (Pi Zero 2 W)
$ sudo -E bash scripts/build-firmware.sh
$ sudo -E bash scripts/build-kernel.sh
$ sudo -E bash scripts/build-image.sh
$ sudo dd if=dist/mintkit-pizero2w.img of=/dev/sdX bs=4M status=progress
